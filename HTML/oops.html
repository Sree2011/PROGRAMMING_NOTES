Object Oriented Programming

The main features of object oriented programming are:

  • Class and Object
  • Abstraction
  • Encapsulation
  • Polymorphism
  • Inheritance
1. Class and Object:

Any entity that has state and behavior is known as an object. For example, Xiaomi, Motorola, Samsung,Apple etc. It can be physical or logical.

The collection of objects is called a class. The objects having similar properties are grouped under a class. Eg: Mobile Phone

2. Abstraction:

Hiding Internal details and showing only what is necessary is called abstraction.

3. Encapsulation:

Binding class and data together is called encapsulation. A class is an example of encapsulation, if all the data members in the class are private.

4. Polymorphism:

Doing same thing in different ways is called polymorphism. It can be achieved by method overriding or method overloading.

5. Inheritance:

When one object acquires the properties of another object, it is known as inheritance. The former is called child object and latter is called parent object.

The C programming is a procedure-oriented language. It does not support classes and objects.
