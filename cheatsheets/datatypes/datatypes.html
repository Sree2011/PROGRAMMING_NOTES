Datatypes

An attribute that identifies a piece of data and instructs a computer system on how to interpret its value is called a data type.

Common datatypes in programming:

  1. Primitive Datatypes
  2. Composite Datatypes
  3. User-defined Datatypes
1. Primitive datatypes:

Primitives are the simplest data types available in a programming language, like numbers or text. They serve as the building blocks for more complex data types. Every programming language has some form of these basic types to handle simple values.

2. Composite Datatypes:

Composite data types are made up of two or more primitive data types. They allow us to group different pieces of data together, like a contact with a name and phone number.

3. User-defined datatypes:

User-defined data types are custom data types that programmers create to represent more complex data. They are tailored to fit the specific needs of the program, like defining a ‘Car’ with properties like color and model.
