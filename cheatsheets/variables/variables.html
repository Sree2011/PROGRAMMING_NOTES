Variables

  • Variables act as storage containers.
  • Variables store values that can be accessed and manipulated throughout the program.
  • In statically typed languages like C and Java, you declare the data type explicitly when defining a variable.
  • In dynamically typed languages like Python, you donâ€™t need to specify the data type explicitly during variable definition.

There are 3 aspects of defining a variable:

  1. Variable Declaration
  2. Variable Definition
  3. Variable Initialization
1. Variable Declaration:

Variable declaration tells the compiler about the existence of the variable with the given name and data type.

2. Variable Definition:

variable definition refers to the declaration of a variable, specifying its name and the type of data it will hold.

3. Variable Initialization:

Variable initialization is the process where the variable is assigned an initial value.

Variable Declaration:

Syntax:

```pseudocode DATATYPE var_name; //for statically typed languages var_name; //for dynamically typed languages ```

Coding examples:

C programming Java Python
```c int var1_name; char var2_name; ``` ```Java int var1_name; char var2_name; ``` ```Python var1_name var2_name ```

Variable Definition and initialisation:

Syntax:

```pseudocode DATATYPE var_name = value; //for statically typed languages var_name = value; //for dynamically typed languages ```

Coding examples:

C programming Java Python
```c int var1_name = 10; char var2_name = 'c'; ``` ```Java int var1_name = 10; char var2_name = 'ch'; ``` ```Python var1_name = 10 var2_name = 'ch' ```
